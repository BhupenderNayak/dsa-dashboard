# 📊 LeetSight Dashboard

LeetSight is a professional-grade web application designed to help software engineers navigate the overwhelming world of technical interview preparation. By aggregating and analyzing thousands of real-world interview questions, LeetSight provides actionable insights and personalized study roadmaps.

## 🚀 Live Demo
[https://leetsight-delta.vercel.app](https://leetsight-delta.vercel.app)

## ✨ Key Features
* **Smart Search:** Instant lookup across 1,500+ tech companies using a high-performance searchable interface (`react-select`).
* **Pattern Analytics:** Visualizes the most frequently tested Data Structure & Algorithm patterns for any company, helping users prioritize their study time.
* **Dynamic Roadmap Generator:** Algorithmically generates custom day-by-day study plans based on the user's availability and target problem frequency.
* **Modern SaaS UI:** Built with a premium Indigo/Violet palette, utilizing Tailwind CSS for a responsive, clean, and professional user experience.

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite
* **UI Components:** React-Select, Tailwind CSS
* **Deployment:** Vercel
* **API Integration:** RESTful architecture using native fetch

## 🚦 Local Setup
1. Clone the repo: `git clone https://github.com/BhupenderNayak/dsa-dashboard.git`
2. Install dependencies: `npm install`
3. Configure the backend URL in `App.jsx` (defaults to production API).
4. Run: `npm run dev`
